# Code of Conduct

## Our Standard

Please participate in a way that is:

1. respectful
2. direct
3. constructive
4. focused on the work

## Unacceptable Behavior

Examples include:

1. harassment or personal attacks
2. discriminatory language
3. intimidation
4. bad-faith disruption of project discussion
5. publishing private information without permission

## Enforcement

Project maintainers may remove comments, issues, or contributions that violate this code of conduct.

Repeated or severe violations may result in a temporary or permanent ban from project spaces.

## Reporting

If you need to report a conduct issue, open a private security-style contact path through the repository maintainers rather than escalating in public discussion.
